Export Annotations, CSV

CoralNet will prepare the annotations as a CSV (comma-separated values) file, which can then be downloaded to your computer and opened in a spreadsheet program like Excel.

The CSV file will include the following columns:

  • Name: The name of the image that the annotation is for.
  • Row: Pixel y-position from the top edge of the image.
  • Column: Pixel x-position from the left edge of the image.
  • Label code or Label ID: Label applied to the point.

The Label format option determines whether labels are written as labelset-defined short codes, or as site-wide label ID numbers. Short codes are more human-readable, while ID numbers can be useful because they are source-independent and very rarely change.

There are also optional sets of columns available:

  • Annotator info: The username of the user who made the annotation, and the date the annotation was made. Imported annotations show as "Imported" and machine annotations show as "robot".
  • Machine suggestions: Labels and confidence scores for the machine's top 5 suggestions.
  • Image metadata - date and auxiliary fields: The image's date field and 5 auxiliary metadata fields.
  • Image metadata - other fields: Other image metadata fields such as depth, camera, and white balance.

Tip: If you are only interested in Confirmed annotations, then first do an image search with "Annotation status" set to "Confirmed", and then export annotations.
